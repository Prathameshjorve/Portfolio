# Portfolio Website Development Plan

## Files to Create:
1. **index.html** - Main HTML structure with all sections (Hero, About, Skills, Projects, Certificates, Contact)
2. **style.css** - Complete styling with dark theme, neon gradients, 3D effects, animations, and responsive design
3. **script.js** - JavaScript for 3D cube animation, smooth scroll, form handling, card animations, and toggle functionality

## Sections Structure:
- Hero: 3D rotating cube (Three.js), name, tagline, CTA buttons
- About: Bio text, 3D avatar, floating tech icons
- Skills: 11 skill cards with glow effects (Python, C++, Java, HTML/CSS/JS, React, MongoDB, ML, DL, Gen AI, Prompt Engineering, Git/GitHub)
- Projects: 4 project cards with hover animations (Wireless EV, AI Assistant, Face Emotion, Portfolio v1)
- Certificates: 3 placeholder certificate cards with 3D tilt effect
- Contact: Form (name, email, message) + social media icons (LinkedIn, GitHub, Instagram, Email)
